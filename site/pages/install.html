Installation

keyhog is a single Rust binary. Install it with cargo, download a prebuilt release, or build from source. Works on Linux, macOS (Intel + Apple Silicon), and Windows with zero configuration.

cargo install

The simplest path. Includes SIMD, ML, entropy, decode-through, and multiline reassembly.

$ cargo install keyhog

Requires Rust 1.89+ and a C++ compiler (for the Hyperscan FFI). On Debian/Ubuntu install build-essential cmake ragel libboost-dev first; on macOS xcode-select --install is enough.

With GPU acceleration

$ cargo install keyhog --features gpu

Pulls the vyre GPU runtime + wgpu drivers. On NVIDIA you get zero-copy NVMe → VRAM DMA; on macOS/Linux without an NVIDIA card you get Metal/Vulkan compute via wgpu. See Performance →.

Minimal (no SIMD, no ML)

$ cargo install keyhog --no-default-features --features regex-engine

Drops Hyperscan, ML scoring, and decode-through. ~3 MB smaller binary, ~10× slower per pattern, no entropy/ML gating. Use this only for embedded or locked-down build environments where pulling Hyperscan is a non-starter.

Prebuilt binaries

Every tagged release attaches platform binaries:

  • keyhog-x86_64-unknown-linux-gnu.tar.gz
  • keyhog-aarch64-apple-darwin.tar.gz (coming soon to the matrix)
  • keyhog-x86_64-pc-windows-msvc.zip
$ curl -L https://github.com/santhsecurity/keyhog/releases/latest/download/keyhog-x86_64-unknown-linux-gnu.tar.gz \
    | tar -xz
$ sudo install keyhog /usr/local/bin/

Docker

$ docker run --rm -v "$PWD:/src" santhsecurity/keyhog scan /src

The image is built from the Dockerfile at the repo root. It bundles the full SIMD + ML feature set, no GPU. ~110 MB compressed.

From source

$ git clone https://github.com/santhsecurity/keyhog.git
$ cd keyhog
$ cargo install --path crates/cli

Local development: cargo run -p keyhog -- scan . from the repo root runs the unmodified binary against your tree.

Verify the install

$ keyhog --version
keyhog v0.5.16

$ keyhog backend
keyhog v0.5.16 | 16 cores | SIMD: AVX-512 | Hyperscan | 891 detectors
  routing matrix:
    < 64 MiB         → simd-regex (Hyperscan)
    ≥ 64 MiB, >2k pat → gpu-zero-copy
    ≥ 256 MiB        → gpu-zero-copy
    no SIMD          → cpu-fallback

Shell completions

# bash
$ keyhog completion bash > /etc/bash_completion.d/keyhog

# zsh
$ keyhog completion zsh > "${fpath[1]}/_keyhog"

# fish
$ keyhog completion fish > ~/.config/fish/completions/keyhog.fish

Upgrading

$ cargo install keyhog --force

Or download a fresh release tarball. Baselines, ignore files, and the incremental cache are forward-compatible across patch releases; major version bumps (0.X) call out any migration in the CHANGELOG.
